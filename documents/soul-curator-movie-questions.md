# SoulCurator - 电影偏好分析问题集

## 问题设计原则
1. 每个问题4个选项
2. 选项覆盖不同情感/审美维度
3. 例子全部使用电影
4. 为AI分析提供结构化数据

## 5个静态问题（选择"电影"后）

### 问题1：情感共鸣深度
**问题**：你希望在电影中找到什么样的情感共鸣？

**选项**：
1. **深刻的人生反思**
   - 例子：《星际穿越》的父女时空之爱
   - 例子：《活着》在历史洪流中的坚韧
   - 特点：引发对生命、时间、存在的深层思考

2. **紧张的智力挑战**
   - 例子：《盗梦空间》的多层梦境逻辑
   - 例子：《记忆碎片》的逆向叙事拼图
   - 特点：需要观众主动解谜，参与叙事建构

3. **温暖的心灵治愈**
   - 例子：《心灵捕手》的心理成长与救赎
   - 例子：《当幸福来敲门》的父爱奋斗
   - 特点：给予希望、温暖和情感慰藉

4. **黑暗的人性探索**
   - 例子：《小丑》的社会边缘人悲剧
   - 例子：《教父》的权力与家族宿命
   - 特点：直面人性阴暗面，引发道德思考

### 问题2：主题关注焦点
**问题**：你更关注哪种主题的深度探讨？

**选项**：
1. **人性与道德的边界**
   - 例子：《辛德勒的名单》在极端环境下的善恶选择
   - 例子：《七宗罪》对人性原罪的现代诠释
   - 特点：探讨人性本质、道德困境、善恶界限

2. **科技与伦理的冲突**
   - 例子：《黑客帝国》的真实与虚拟边界
   - 例子：《她》中人与AI的情感关系
   - 特点：反思技术进步对人类社会的冲击

3. **社会与个体的关系**
   - 例子：《寄生虫》的阶级固化与生存挣扎
   - 例子：《熔炉》对体制腐败的控诉
   - 特点：关注社会结构、权力关系、个体命运

4. **存在与虚无的哲思**
   - 例子：《2001太空漫游》的人类进化与宇宙意义
   - 例子：《银翼杀手》复制人的人性追问
   - 特点：探讨生命意义、存在本质、宇宙观

---

### 问题3：叙事艺术偏好
**问题**：你欣赏哪种叙事艺术的表达？

**选项**：
1. **经典的三幕式结构**
   - 例子：《肖申克的救赎》的希望与救赎之旅
   - 例子：《阿甘正传》通过个人史反映时代
   - 特点：起承转合清晰，情感积累自然

2. **多线交织的复杂叙事**
   - 例子：《低俗小说》的非线性碎片重组
   - 例子：《通天塔》的全球化命运交织
   - 特点：多条线索并行，最终汇聚成整体

3. **时间跳跃的心理叙事**
   - 例子：《蝴蝶效应》的时空修改与因果链
   - 例子：《降临》的非线性时间感知
   - 特点：打破时间顺序，强调心理真实

4. **沉浸式的角色叙事**
   - 例子：《少年派的奇幻漂流》的主观生存史诗
   - 例子：《美丽人生》父亲用游戏保护儿子
   - 特点：强烈代入感，情感沉浸深度

---

### 问题4：视觉美学倾向
**问题**：哪种视觉风格更能触动你？

**选项**：
1. **宏大的史诗构图**
   - 例子：《指环王》的中土世界全景
   - 例子：《阿拉伯的劳伦斯》的沙漠史诗
   - 特点：广阔空间感，历史厚重感

2. **精致的色彩美学**
   - 例子：《布达佩斯大饭店》的对称与粉色调
   - 例子：《英雄》的色彩叙事与意境
   - 特点：色彩作为叙事语言，视觉和谐

3. **实验性的镜头语言**
   - 例子：《公民凯恩》的景深镜头与叙事
   - 例子：《重庆森林》的手持摄影与都市脉搏
   - 特点：打破常规，探索电影语言边界

4. **震撼的视觉特效**
   - 例子：《阿凡达》的潘多拉生态奇观
   - 例子：《盗梦空间》的城市折叠与零重力
   - 特点：技术创造不可能，拓展想象边界

---

### 问题5：角色灵魂深度
**问题**：你希望角色拥有怎样的灵魂深度？

**选项**：
1. **复杂矛盾的多面性**
   - 例子：《教父》迈克尔从理想青年到黑帮教父
   - 例子：《黑天鹅》妮娜的完美主义与自我毁灭
   - 特点：角色内心冲突，善恶交织

2. **漫长曲折的成长弧**
   - 例子：《心灵捕手》威尔从防御到敞开心扉
   - 例子：《国王的演讲》乔治六世克服口吃
   - 特点：角色在困境中蜕变成长

3. **悲剧宿命的反英雄**
   - 例子：《小丑》亚瑟从被欺凌者到混乱象征
   - 例子：《这个杀手不太冷》里昂的温柔与宿命
   - 特点：非传统英雄，悲剧性命运

4. **平凡真实的人性光**
   - 例子：《当幸福来敲门》克里斯加德纳的父爱奋斗
   - 例子：《海边的曼彻斯特》李无法走出的悲伤
   - 特点：普通人的坚韧，真实情感力量

## AI提示词设计（基于这5个问题）

### 阶段1：初步画像生成
```
你是一个电影分析专家。基于用户的以下选择：

1. 情感共鸣偏好：{用户选择1}
   - 例子：{对应电影例子}

2. 主题关注焦点：{用户选择2}
   - 例子：{对应电影例子}

3. 叙事艺术偏好：{用户选择3}
   - 例子：{对应电影例子}

4. 视觉美学倾向：{用户选择4}
   - 例子：{对应电影例子}

5. 角色灵魂深度：{用户选择5}
   - 例子：{对应电影例子}

请生成一个用户电影偏好的初步画像，包括：
- 核心情感需求分析
- 主题兴趣图谱（权重分布）
- 审美倾向特征
- 潜在推荐方向（3-5个关键词）
```

### 阶段2：动态问题生成（AI介入）
```
基于用户的初步电影偏好画像：
{初步画像内容}

请生成3个个性化的深入问题，帮助进一步细化用户的电影偏好。
每个问题应有4个选项，覆盖用户可能感兴趣但未明确表达的维度。

要求：
1. 问题要基于用户已表现出的偏好倾向
2. 选项要有具体的电影例子支撑
3. 问题间要有逻辑递进关系
```

### 阶段3：最终推荐生成
```
基于用户的完整电影偏好画像：
{完整画像内容}

请推荐5部最适合用户的电影，每部推荐包含：

1. **电影名称** + 导演 + 年份
2. **核心匹配点**（与用户偏好的具体匹配）
3. **情感共鸣分析**（为什么这部电影能触动用户）
4. **观看建议**（最佳观看状态、注意事项）
5. **延伸思考**（这部电影可能引发的思考方向）

要求推荐具有多样性，覆盖用户偏好的不同侧面。
```
