## 🧠 技术决策与学习

### 关键决策
1. **Java 17升级**：放弃Java 8，拥抱现代Java生态
2. **PostgreSQL 14降级**：兼容旧版Navicat，务实优先
3. **JavaScript选择**：放弃TypeScript，追求开发速度
4. **最小化后端**：先跑通再优化，MVP思维
5. **Docker仅用于数据库**：后端本地运行，便于调试

### 遇到的问题与解决方案
1. **问题**: PostgreSQL 15 SCRAM认证不兼容Navicat 12
   **解决**: 降级到PostgreSQL 14，清理数据卷重新初始化

2. **问题**: JPA查询语法错误
   **解决**: 注释复杂查询，创建简化Repository

3. **问题**: Spring Security默认开启认证
   **解决**: 创建SecurityConfig完全禁用（开发环境）

4. **问题**: 端口8080冲突  
   **解决**: 使用不同端口或关闭占用进程

5. **问题**: TypeScript学习成本 vs 开发速度
   **解决**: 迁移到纯JavaScript，保持开发节奏

	let last = 0

 		const now = new Date().now()
    if ((now - last) < delay) {

    }

  }
}
