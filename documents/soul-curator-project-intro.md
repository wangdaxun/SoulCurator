# SoulCurator（灵魂策展人）项目介绍
## 一句话说清楚：用选择题帮你发现文艺灵魂的网站

---

## 🎯 项目是干啥的？

想象一下：你玩过《云顶之弈》的海克斯强化选择吗？SoulCurator就是**用那种游戏化的选择题，帮你探索自己的文艺灵魂**。

### 简单来说：
1. **你回答几道选择题**（比如：电影、文学、音乐、游戏，你更亲近哪个？）
2. **系统生成你的"灵魂画像"**（一个代表你文艺偏好的抽象图形）
3. **获得个性化推荐**（不是"猜你喜欢"，是"懂你灵魂"的深度作品）

### 就像这样：
```
进入网站 → 选择"开始灵魂探索" → 回答5道选择题 → 看到你的灵魂画像 → 获得专属推荐
```

---

## 🔍 解决什么问题？

### 问题1：审美不平等
有些人从小看《星际穿越》，有些人只看过《满江红》。  
这不是谁更"高级"，而是**机会不平等**——很多人根本没机会接触深度作品。

### 问题2：算法茧房
抖音、B站推荐的都是"刺激"，不是"深度"。  
你被困在算法里，接触不到真正能触动灵魂的作品。

### 问题3：孤独探索
你被《最后的生还者》震撼，但身边没人懂。  
想探索更深的文艺世界，但不知道从哪里开始。

---

## 🎮 怎么解决的？

### 方案1：游戏化降低门槛
- 不是让你写论文，而是**做选择题**
- 像玩游戏一样探索自我
- 5分钟就能完成第一次灵魂探索

### 方案2：可视化结果
- 你的文艺偏好变成一个**酷炫的灵魂画像**
- 看得见自己的"灵魂模样"
- 分享给朋友："看，这就是我的文艺灵魂"

### 方案3：个性化策展
- 基于你的灵魂画像，推荐**真正适合你的作品**
- 不是"热门推荐"，是"灵魂匹配"
- 每周更新，跟着你的成长一起变化

## ❓ 常见问题

### Q：这和豆瓣有什么区别？
A：豆瓣是"大家评价"，SoulCurator是"帮你发现"。豆瓣告诉你《星际穿越》评分高，我们告诉你"为什么《星际穿越》适合你"。

### Q：为什么要付费？
A：免费功能足够体验。付费是为了深度服务：详细分析、专属推荐、成长追踪。就像健身房：可以免费试用，但私教课要付费。

### Q：我的数据安全吗？
A：绝对安全。我们只分析你的文艺偏好，不收集隐私信息。所有数据匿名处理，符合中国网络安全法。

### Q：如果我不喜欢文艺怎么办？
A：那可能不适合你。但很多人以为自己"不喜欢文艺"，其实只是没找到适合自己的作品。试试看，也许有惊喜。

### Q：这个能赚钱吗？
A：我们的目标是：先帮助用户，再获得合理回报。如果10万人中5%付费（5000人），每人每年花200元，就是100万收入。足够养活小团队，继续做好产品。
