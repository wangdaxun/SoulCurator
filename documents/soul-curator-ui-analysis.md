## 第二阶段：艺术形式选择后的流程设计

### 选择"电影"后的5个静态问题设计

**问题设计原则**：
1. 每个问题有4个选项
2. 选项覆盖不同情感维度
3. 问题间有逻辑递进关系
4. 为AI分析提供足够数据

### 5个静态问题草案：

#### 问题1：情感基调选择
**问题**：你希望电影带给你什么样的情感体验？
- [ ] 深刻反思（如《星际穿越》）
- [ ] 紧张刺激（如《盗梦空间》）
- [ ] 温暖治愈（如《心灵捕手》）
- [ ] 黑暗悬疑（如《白夜行》风格）

#### 问题2：主题偏好
**问题**：你对哪种主题更感兴趣？
- [ ] 人性探索（如《底特律变人》）
- [ ] 科技伦理（如《黑镜》）
- [ ] 社会现实（如《活着》）
- [ ] 哲学思辨（如《三体》）

#### 问题3：叙事风格
**问题**：你偏好哪种叙事方式？
- [ ] 线性叙事（传统故事）
- [ ] 多线交织（如《十三机兵防卫圈》）
- [ ] 非线性时间（如《记忆碎片》）
- [ ] 沉浸式体验（如《最后的生还者》）

#### 问题4：视觉美学
**问题**：你更看重电影的哪个方面？
- [ ] 画面构图（如《奥本海默》）
- [ ] 色彩运用（如《布达佩斯大饭店》）
- [ ] 镜头语言（如《公民凯恩》）
- [ ] 特效制作（如《阿凡达》）

#### 问题5：角色深度
**问题**：你希望角色有什么特点？
- [ ] 复杂多面（如《教父》）
- [ ] 成长弧光（如《肖申克的救赎》）
- [ ] 反英雄特质（如《小丑》）
- [ ] 普通人共鸣（如《当幸福来敲门》）

## 第四阶段：AI介入时机与提示词设计

### AI介入节点
1. **静态问题完成后**：基于5个问题的答案生成初步画像
2. **动态问题阶段**：根据初步画像生成个性化问题
3. **推荐生成阶段**：基于完整画像生成推荐

### 提示词设计框架

#### 阶段1：初步画像生成提示词
```
你是一个文艺作品分析专家。基于用户的以下选择：
1. 情感基调：{用户选择}
2. 主题偏好：{用户选择}
3. 叙事风格：{用户选择}
4. 视觉美学：{用户选择}
5. 角色深度：{用户选择}

请生成一个用户文艺偏好的初步画像，包括：
- 核心情感需求
- 主题兴趣图谱
- 审美倾向分析
- 潜在推荐方向
```

#### 阶段2：动态问题生成提示词
```
基于用户的初步画像：
{初步画像内容}

请生成3个个性化的深入问题，帮助进一步细化用户的文艺偏好。
每个问题应有4个选项，覆盖用户可能感兴趣但未明确表达的维度。
```

#### 阶段3：最终推荐生成提示词
```
基于用户的完整偏好画像：
{完整画像内容}

请推荐5部最适合用户的电影，每部推荐包含：
1. 电影名称
2. 推荐理由（与用户偏好的匹配点）
3. 情感共鸣点
4. 观看建议
```
