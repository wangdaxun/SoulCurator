# SoulCurator UI设计需求文档

## 项目概述
**项目名称**: SoulCurator（灵魂策展人）
**项目定位**: 基于个人文艺兴趣的智能推荐系统
**目标用户**: 深度文艺爱好者，追求情感共鸣和审美深度
**设计风格**: 暗黑文艺风，科技感与艺术感融合

## 设计参考
### 已完成的页面
- **入口页面** (`EntryGuide.vue`): 艺术形式选择页面
- **视觉风格**: 暗黑背景 + 粒子效果 + 发光卡片 + 破碎动画
- **色彩方案**: 深色主调，紫色/蓝色/金色/绿色高亮
- **交互特点**: 卡片选择、碎裂动画、淡入淡出

### 设计语言延续
- 主背景色: `#050507` (接近纯黑)
- 文字颜色: 白色不同透明度层级
- 发光效果: 柔和光晕，颜色主题相关
- 动画风格: 平滑过渡，破碎效果，粒子运动

## 需要设计的页面/组件

### 页面1：问题流程主页面

#### 页面结构
```
┌─────────────────────────────────────────────────┐
│ 进度条 (顶部固定)                                │
├─────────────────────────────────────────────────┤
│                                                  │
│           问题卡片区域                           │
│   (居中，最大宽度1200px)                         │
│                                                  │
├─────────────────────────────────────────────────┤
│ 导航按钮 (上一题/下一题)                         │
└─────────────────────────────────────────────────┘
```

#### 进度条设计需求
- **位置**: 页面顶部，固定定位
- **内容**: 7个步骤的进度指示
- **步骤**:
  1. 🚪 灵魂之门 (已选择艺术形式)
  2. 💖 情感共鸣 (问题1)
  3. 🎯 主题焦点 (问题2) 
  4. 📖 叙事艺术 (问题3)
  5. 🎨 视觉美学 (问题4)
  6. 👤 角色灵魂 (问题5)
  7. 🔮 生成画像 (结果页面)

- **状态设计**:
  - 已完成: 实心图标 + 连接线
  - 当前进行: 发光脉冲效果
  - 未完成: 半透明灰色
  - 连接线: 微光流动动画

- **交互反馈**:
  - hover时显示步骤名称
  - 点击可跳转到已完成步骤
  - 进度更新时有数字滚动动画

#### 视觉设计要求
1. **卡片容器**:
   - 背景: 半透明毛玻璃效果
   - 边框: 1px发光边框，颜色随问题变化
   - 阴影: 多层阴影创造深度感
   - 圆角: 16px-24px

2. **问题区域**:
   - 标题字体: Serif字体，较大字号，字距宽松
   - 序号显示: 右上角徽章样式
   - 问题描述: 清晰易读，适当留白

3. **选项区域**:
   - 布局: 2x2网格 (桌面) / 1x4列表 (移动)
   - 间距: 选项间有足够呼吸空间
   - 状态指示: 选中状态明显区分

#### 交互状态设计
1. **默认状态**:
   - 选项卡片: 半透明，轻微发光
   - 边框: 细线，低透明度

2. **悬停状态**:
   - 轻微放大 (scale: 1.02)
   - 发光增强
   - 添加微动效

3. **选中状态**:
   - 背景变亮
   - 边框发光脉冲
   - 添加选中标记 (✓图标)
   - 破碎玻璃效果动画

4. **禁用状态**:
   - 降低透明度
   - 去除交互效果

### 页面4：导航按钮设计

#### 按钮类型
1. **上一题按钮**:
   - 图标: ← 箭头
   - 状态: 第一题时禁用
   - 样式: 轮廓按钮

2. **下一题按钮**:
   - 图标: → 箭头 (最后一题时变为"完成")
   - 状态: 未选择答案时禁用
   - 样式: 实心按钮，发光效果

3. **完成按钮**:
   - 文案: "生成我的灵魂画像"
   - 样式: 特殊样式，强烈召唤行动

#### 按钮状态
- 默认: 半透明，轻微发光
- 悬停: 发光增强，轻微放大
- 激活: 点击反馈，波纹效果
- 禁用: 低透明度，无交互

### 页面5：响应式设计规范

#### 断点设置
- **桌面端** (≥1200px): 完整布局
- **平板端** (768px-1199px): 适配布局
- **移动端** (<768px): 简化布局

#### 移动端适配
1. **进度条**: 简化显示，只显示当前步骤
2. **问题卡片**: 全屏宽度，减少边距
3. **选项布局**: 垂直排列，1列
4. **导航按钮**: 底部固定，全宽按钮

### 问题2：主题关注焦点
**问题**: 你更关注哪种主题的深度探讨？

4. 存在与虚无的哲思 (《2001太空漫游》、《银翼杀手》)

### 问题3：叙事艺术偏好
**问题**: 你欣赏哪种叙事艺术的表达？
